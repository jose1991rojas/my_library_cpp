# my_library_cpp
Custom standard library for C++ with string manipulation functions from older standards; however, I will update my library (there will be other utilities) as time goes on.

Mi libreria C++ con funciones de manipulación de cadenas de viejos estándares, la ire actualizando (existiran otras utilidades) a media que avance  en el tiempo



Installation (Manual Installation)
Clone (git clone) the repository into your path: /home/user/

Copy the library to the system path: sudo cp ruzza.h /usr/local/include/
or mv /home/user/ruzza.h /usr/local/include/ruzza.h

Installation (Automatic Installation)
<img width="415" height="189" alt="image" src="https://github.com/user-attachments/assets/bf53bb49-4282-4055-837c-d8cc29136e9b" />
Hay un instalador (Bash script) de la libreria que se encarga de colocar el archivo en la ruta correcta del sistema, para 
que el compilador reconozca a 'ruzza.h'.

Instrucciones de uso en la terminal linux:

1.- Nombre del instalador automatico: 
    'installer.sh'

2.- Instruccion de uso, otorgar permisos de ejecucion: 
    chmod +x installer.sh

3.- Ejecucion:
    ./installer.sh

-

There is an installer (Bash script) for the library that places the file in the correct system path so that the compiler recognizes 'ruzza.h'.

Instructions for use in the Linux terminal:

1.- Name of the automatic installer:

    'installer.sh'

2.- Instruction to grant execution permissions:

    'chmod +x installer.sh'

3.- Execution:

    './installer.sh'
